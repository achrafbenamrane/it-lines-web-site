# IT Lines - Digital Solutions & IT Consultancy Website

A sleek, professional, and high-converting website for IT Lines - connecting businesses to digital success through innovative technology solutions. Built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

### Design & Aesthetics
- **Modern & Clean**: Corporate design with tech-forward elegance
- **Responsive**: Fully responsive across all devices
- **Interactive**: Smooth animations and micro-interactions
- **Professional**: High-quality UI components with attention to detail

### Sections Included

1. **Header/Navigation** - Fixed header with scroll effect, responsive menu, language selector
2. **Hero Section** - Split-screen layout with animated statistics and CTAs
3. **Services Section** - 6 service cards with hover effects (Web Dev, Marketing, E-commerce, Cloud, Security, Software)
4. **Why Choose Us** - Three feature columns with statistics and process timeline
5. **About/Team** - Animated counters (500+ projects, 100+ clients, 50+ experts)
6. **Testimonials** - Interactive carousel with client reviews
7. **Final CTA** - Dark gradient section with prominent call-to-action
8. **Contact** - Modern form with floating labels and contact info
9. **Footer** - Multi-column layout with newsletter subscription

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
